# 深度求索 自动易模

感梁圣开源 深度求索 四修精本 之惠。然每百万词元索直二十四元，余资寡薄，不能久承，故作此插件，以避峰时之费。

值每日高峰涨价之时，本插件将于 大言模 请求未发之前，先易其所用模型：

- 京师时辰自巳初至午末、自未初至酉末：用君所设之备用模型。
- 余时：复用君所设之 深度求索 四修精本。

本插件借 爱思特波特 会话级 坊主 偏好以易模型，不改全局默认模型。

## 安装之法

1. 置插件目录于 爱思特波特 之 `data/plugins/` 中。
2. 重启 爱思特波特，或以插件管理命令重载之。
3. 入 爱思特波特 网页门面，设插件诸项。

## 配置

于插件配置页设之：

| 配置项 | 说明 |
| --- | --- |
| `enabled` | 是否启用自动易模 |
| `deepseek_provider_id` | 非高峰时所用 深度求索 四修精本 坊主 |
| `peak_provider_id` | 高峰时所用备用 坊主 |

二 坊主 皆可由 爱思特波特 网页门面 模型提供商选择器择之。

## 指令

- `/deepseek_switch_status`：观当前京师时辰、所处时段及目标模型。
- `/ds_switch_status`：同上。

## 注意

初入对应时段而有请求时，插件将在此会话内写入 坊主 偏好；其后同会话皆循当前时段所宜之模型。若越高峰之界，则于下一次 大言模 请求时，自动易至新目标模型。
